## MacLev v0.4.4

### Changes
- improves settings tab hit targets so each tab card is fully clickable
- adds explanatory copy for the window floating toggle in General settings
- aligns camera and microphone permission columns for per-site rules
- removes the repeated custom permission popup by delegating to macOS permission state in Ask mode
- refines top toolbar layout so back/forward are in a merged pill and refresh sits in the address bar
- adjusts minimum window size to a more practical browser baseline

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
