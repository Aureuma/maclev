## maclev v0.4.1

### Changes
- reduced the minimum browser window size so the app can be resized much smaller
- added a release runbook under `docs/`
- keeps the current preferences, media permissions, and cask-based app distribution model

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
