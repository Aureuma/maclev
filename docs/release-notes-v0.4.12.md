## MacLev v0.4.12

### Changes
- Fixes the stretched top address row by giving the tab strip and address row explicit heights.
- Keeps the tab strip above the address row while making the address controls compact and properly aligned.
- Preserves the full-width browser chrome behavior from the previous release.

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
