## maclev v0.4.6

### Changes
- Integrates browser controls into the window title bar using toolbar chrome (Safari-style placement).
- Keeps tab strip and navigation/address controls inside a compact toolbar region.
- Reduces default and minimum window dimensions for a tighter interface.
- Tightens control spacing and compactifies address/bar sizing.
- Maintains existing tabbing and browser shortcut behavior from v0.4.5.

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
