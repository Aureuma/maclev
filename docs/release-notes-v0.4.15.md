## MacLev v0.4.15

### Changes
- Makes the flying saucer emoji non-interactive so clicking it no longer toggles the always-on-top switch.
- Keeps the saucer visually next to the switch as a label only.

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
