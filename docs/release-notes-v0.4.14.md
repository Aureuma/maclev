## MacLev v0.4.14

### Changes
- Removes the new-tab `+` button from the address row.
- Keeps the new-tab button only on the far right of the tab bar.
- Preserves the compact top chrome layout from the previous release.

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
