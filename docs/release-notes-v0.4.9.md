## MacLev v0.4.9

### Changes
- Rebuilds the top window chrome to use native toolbar placements instead of a centered floating title view.
- Moves the tab strip below the title bar so it no longer overlaps or fights the macOS window controls.
- Expands the address field across the toolbar and keeps the floating toggle compact on the trailing side.

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
