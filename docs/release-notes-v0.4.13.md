## MacLev v0.4.13

### Changes
- Moves the new-tab button to the far right edge of the tab strip.
- Places the flying saucer emoji next to the always-on-top toggle switch.
- Keeps the compact top chrome sizing from the previous release.

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
