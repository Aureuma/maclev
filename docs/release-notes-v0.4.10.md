## MacLev v0.4.10

### Changes
- Moves the browser controls into a full-width in-window top bar so the address row spans the window instead of being constrained by the native toolbar principal slot.
- Keeps the tab strip below that top bar for a clearer separation between window chrome and tab content.
- Preserves the compact floating toggle and browser control layout while fixing the width behavior.

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
