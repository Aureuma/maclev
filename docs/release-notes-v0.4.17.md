## MacLev v0.4.17

### Changes
- Makes the flying saucer emoji clickable again so it toggles the always-on-top switch.
- Keeps the switch itself functional, with both controls updating the same floating state.

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
