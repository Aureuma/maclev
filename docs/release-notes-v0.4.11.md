## MacLev v0.4.11

### Changes
- Moves the tab strip above the address row so the top chrome reads in the expected order.
- Reduces the address bar and control heights so the browser chrome is more compact.
- Keeps the full-width top bar behavior introduced in the previous release.

### Install
```bash
brew tap aureuma/maclev
brew install --cask maclev
```

### Launch
```bash
open /Applications/maclev.app
```
